# How To Use Backend HAPI
Rename or Configuration file ".env.example" be ".env".

1. Run Npm Install
```
npm i
```

2. Donwload Sqlite3 from
```
https://www.sqlite.org/
```

3. Migration Table
```
npm run migrate
```

4. Start Project
```
npm run start
```

5. Open Link in Locally.
```
http://127.0.0.1:9000
```
